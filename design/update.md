# 《神どろ》备牌库终极卡牌（公共）- v1.0

备牌库（Sideboard）中的卡牌通常不编入 50 张主卡组，主要通过 **13点神抽（Overkill）** 机制被检索并加入战场。它们是针对极端劣势局面的“终极解药”。

## 一、 核心平衡原则
1. **机制重于数值**：拥有“规则级”进场效果的使魔，其基础攻防面板通常设定在较低水平。
2. **高费用限制**：费用设定在 8-10 费。若非通过“13点神抽”获得的费用折扣，玩家极难在常规回合直接从手牌打出。
3. **高风险博弈**：部分效果带有“强制败北”或“资源耗尽”的代价，确保其仅作为绝地反击的最后手段。

## 三、 机制深度说明

### 1. 乌洛波洛斯 (Ouroboros) - 终极时轴
* **战术定位**：纯粹的机制触发器。2/2/2 的极弱面板确保了它无法在当回合提供输出，核心价值在于通过额外回合让你的其他使魔重置攻击频率并回满费用。它是“死线残响”触发后的最佳搭档。

### 2. 米迦勒 (Michael) - 战场重置
* **战术定位**：针对铺场流与陷阱流的究极手段。通过 9 费的高额定价限制其出场率。进场回复生命值的效果能有效缓解对方快攻带来的血量分压力。

### 3. 卡奥斯 (Chaos) - 牌库制裁
* **战术定位**：磨牌流的最后一块拼图。保留 7 张牌的设计精准避开了直接致死的嫌疑，但会让对手瞬间进入“牌库危机（≤5张）”的倒计时，并严重削弱其手牌资源储备逻辑。其自带的护卫和 5 点生命提供了基础的防线。

### 4. 尤斯蒂娅 (Justitia) - 生死易位
* **战术定位**：防守反击流的梦幻卡。10 费意味着它是全游最高代价的卡牌，不通过神抽折扣几乎无法上场。通过互换血量，它能瞬间将你的斩杀线与对手的存活线对调。

# 《神どろ》大魔法系列 (High-Level Magic) - v1.0

大魔法是改变战场结构、强制重置优劣势判定的战略级手段。其费用设定直接影响了发动后剩余的势能结算空间。

## 一、 战场肃清类 (Board Clears)
这类魔法通过大规模破坏使魔，强制改变公式中的“威胁分”与“斩杀线”。

| 魔法名称 | 费用 | 类型 | 效果说明 | 战术逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| **神之愤怒** | **5** | 瞬间 | 破坏场上**所有**使魔。 | **最高效的平场手段**。5费的定价允许玩家在清场后仍有余力下怪或盖陷阱，是控制流的核心。 |
| **万雷天引** | **8** | 瞬间 | 破坏对方场上**所有**使魔。 | **高昂的单向收割**。8费意味着这几乎是当回合的唯一动作，通过牺牲自己的节奏换取对方场面的彻底崩塌。 |

---

## 二、 魔法/陷阱拆除类 (Anti-Support)
针对持续魔法提供的稳定“威胁值”和触发魔法带来的“未知风险”。

| 魔法名称 | 费用 | 类型 | 效果说明 | 战术逻辑 |
| :--- | :--- | :--- | :--- | :--- |
| **虚无结界** | **3** | 瞬间 | 破坏场上**所有**持续魔法。 | **低成本的势能重置**。当双方都在通过持续魔法堆叠威胁值时，3费可以瞬间让这些“虚假繁荣”归零。 |
| **净化之光** | **5** | 瞬间 | 破坏对方场上**所有**持续魔法。 | **单向势能打击**。支付更高的代价以保留自己的持续增益，同时剥离对方的威胁分贡献。 |
| **真实的探照灯**| **3** | 瞬间 | 破坏对方场上**所有**盖伏的触发魔法（陷阱）。 | **进攻冲锋号**。在发起总攻或宣告“跳脸”前，用 3 费排除所有被强行中断的风险。 |

## ⚖️ 设计笔记与系统联动建议

1. **“神之愤怒”与威胁值的博弈**：
   由于《神どろ》中使魔带有威胁值，5费清场不仅是清除了伤害，更是瞬间抹平了对方的“势能分”。如果玩家在清场后能拍下一个低费高威胁使魔，能瞬间将局势从“极劣”转为“均势”甚至“小优”。

2. **“万雷天引”的释放时机**：
   8费在常规回合非常沉重，但在 **13点神抽（Overkill）** 带来的费用折扣下，这张卡可能仅需 3-4 费即可发动。这种“打折后的单向清场”将是游戏中最恐怖的逆转爆发点。

3. **陷阱拆除的重要性**：
   考虑到你设计的“死线残响”等高风险强力陷阱，3费的“探照灯”是所有快攻和中速卡组必须满编 3 张的博弈牌，否则很容易在临门一脚时被强制翻盘。

   # 《神どろ》使魔核心词条效果速查表 (v1.0)

本表列出了《神どろ》中使魔所携带的所有常驻与触发类词条的具体执行效果。

## 一、 基础战斗类词条
| 词条名称 | 游戏内执行效果 (Effect) |
| :--- | :--- |
| **护卫** | 只要拥有此词条的使魔在场，敌方必须先指定击杀该使魔，**无法直接攻击你的角色或其他无“护卫”的使魔**。 |
| **疾风** | 拥有此词条的使魔在从手牌召唤下场的**当回合，即可立刻发起攻击指令**。 |
| **连击** | 拥有此词条的使魔在单回合内，可以执行**两次攻击指令**。 |
| **回复** | 在你的回合结束阶段（或开始阶段），该使魔为自身**回复固定数值的 HP**（未标注则默认为 1 点）。 |
| **魔抗** | 该使魔在场上时，**无法被敌方的任何“魔法卡”和“使魔效果”指定为目标**。（但依然可以受到全体范围魔法的影响或被使魔物理攻击）。 |

---

## 二、 机制互动与高阶词条
| 词条名称 | 游戏内执行效果 (Effect) |
| :--- | :--- |
| **吸血** | 该使魔在战斗中对敌方（无论角色还是使魔）造成伤害时，**为你的角色回复等同于伤害数值的 HP**。 |
| **潜行** | 在该使魔**主动发起第一次攻击之前**，它不能被敌方使魔设为攻击目标，也不能被敌方的魔法卡指定。一旦发起攻击，潜行状态解除。 |
| **死斗** (必杀) | 无论该使魔的攻击力数值是多少，只要它在战斗中**对敌方使魔造成了伤害（哪怕只有 1 点），敌方使魔直接被破坏**进入墓地。 |
| **遗言** | 当该使魔的 HP 降为 0 或被效果**破坏送入墓地时，触发写在卡牌上的专属效果**（如抽牌、造成伤害等）。 |
| **威压** | 只要该使魔在场上存活，敌方场上所有使魔在结算时，其面板基础**“威胁值”减半**（向下取整）。 |
| **看破** | 只要该使魔在场上，对方盖伏的“触发魔法（陷阱）”的**触媒条件对你公开**；且该使魔发起攻击时，不会触发敌方盖伏的触发魔法。 |
